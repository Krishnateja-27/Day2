Browser JS vs Node js
1.JavaScript is a programming language that is used for writing the
scripts on the website while Node Js is javascript run time
environment
2.JavaScript can only be run in the browsers while NodeJs can run
outside the browser
3.BrowserJs basically used on the client side while Node JS is mostly
used on the server side
4.JS is capable enough to add HTML and play with DOM while
NodeJs does not have the capability to add HTML tags
5.Js can run in any browser engine while NodeJs can only run in V8
engine of Google Chrome
